Ciclo 1: Cadastro de usuário

História de usuário: "Quero me cadastrar e acessar o mural"

Critérios de aceitação:

- O email deve ser único.
- A senha deve ser criptografada.
- O sistema deve exibir uma mensagem de sucesso ou de erro.

Ciclo 2: Postagem

História: Quero criar uma publicação como usuário logado.

Critérios de aceitação:

- Não permitir postagem nula.
- Associar a postagem ao usuário logado.

Ciclo 2: Curtir publicação

História: Quero curtir uma publicação como usuário logado.

Critérios de aceitação:

- Impedir curtidas duplicadas.
- Atualizar contador de likes.
